# Project Brief

## Template / Product Name
Podcast Website Template

## Niche / Industry
Media — Podcast

## Target Customer / Buyer
Independent podcasters, creators, and media brands

## Site Type
One-page show discovery and subscriber growth site

## Goal of the Website
Grow listeners, subscribers, and sponsor credibility

## Pages Required
- Home

## Required Sections
- Hero
- About the Show
- Featured Episodes
- Host
- Testimonials / Reviews
- CTA
- Contact

## Optional Sections
- Sponsors
- Newsletter
- Guest Application

## Visual Direction
Modern, editorial, personality-led

## Content Direction
Conversational, energetic, audience-focused

## Functional Requirements
- Subscription/listen buttons
- Featured episode cards
- Contact/partnership form

## Technical Constraints
- Static HTML/CSS/JS only
- No live audio player required
- Optimised cover art and card layouts

## Delivery Format
Downloadable static template pack with organised source files and buyer-facing documentation.

## Notes / Special Instructions
This template should feel immediately relevant to its niche while still following the shared production system and reusable section structure.

## Conversion Strategy
Visitor interest → subscribe / listen now

## Primary User Action
Listen or subscribe

## Secondary Actions
- Browse episodes
- Meet the host
- Contact for partnerships

## Trust Mechanisms
- Episode highlights
- Listener reviews
- Guest or media credibility

## Content Priority Order
1. Show positioning
2. Episode highlights
3. Subscription CTA
