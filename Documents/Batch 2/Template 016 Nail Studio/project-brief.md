# Project Brief

## Template / Product Name
Nail Studio Template

## Niche / Industry
Beauty — Nail Studio

## Target Customer / Buyer
Nail salon owners and independent nail technicians

## Site Type
One-page lead generation

## Goal of the Website
Increase bookings for nail services

## Pages Required
- Home

## Required Sections
- Hero
- Services
- Pricing
- Gallery
- Testimonials
- CTA
- Contact

## Optional Sections
- FAQ
- About
- Gift Cards

## Visual Direction
Clean, stylish, feminine, polished

## Content Direction
Friendly, modern, beauty-focused

## Functional Requirements
- Booking/contact form
- Service and pricing cards
- Responsive gallery layout

## Technical Constraints
- Static HTML/CSS/JS only
- Lightweight gallery interactions only
- Accessible form and contrast

## Delivery Format
Downloadable static template pack with organised source files and buyer-facing documentation.

## Notes / Special Instructions
This template should feel immediately relevant to its niche while still following the shared production system and reusable section structure.

## Conversion Strategy
Service interest → appointment booking

## Primary User Action
Book an appointment

## Secondary Actions
- Browse services
- See pricing
- View gallery

## Trust Mechanisms
- Gallery of work
- Client testimonials
- Service clarity

## Content Priority Order
1. Booking CTA
2. Service menu
3. Visual proof
