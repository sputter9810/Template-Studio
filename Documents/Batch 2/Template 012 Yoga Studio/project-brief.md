# Project Brief

## Template / Product Name
Yoga Studio Template

## Niche / Industry
Wellness — Yoga

## Target Customer / Buyer
Yoga studio owners, yoga teachers, and boutique wellness operators

## Site Type
One-page lead generation

## Goal of the Website
Increase class bookings and intro pass sales

## Pages Required
- Home

## Required Sections
- Hero
- Class Styles
- Schedule
- About
- Testimonials
- CTA
- Contact

## Optional Sections
- FAQ
- Memberships
- Gallery

## Visual Direction
Calm, airy, welcoming, grounded

## Content Direction
Warm, supportive, beginner-friendly

## Functional Requirements
- Sticky nav with anchor links
- Intro booking/contact form
- Responsive timetable section

## Technical Constraints
- Static HTML/CSS/JS only
- No heavy animation libraries
- Optimised imagery and accessible forms

## Delivery Format
Downloadable static template pack with organised source files and buyer-facing documentation.

## Notes / Special Instructions
This template should feel immediately relevant to its niche while still following the shared production system and reusable section structure.

## Conversion Strategy
Intro offer → recurring class attendance

## Primary User Action
Book an intro class

## Secondary Actions
- View timetable
- Explore class styles
- Meet instructors

## Trust Mechanisms
- Instructor credentials
- Student testimonials
- Studio atmosphere

## Content Priority Order
1. Offer (intro pass)
2. Class types
3. Trust and calm experience
