# Project Brief

## Template / Product Name
Electrician Template

## Niche / Industry
Trade Services — Electrician

## Target Customer / Buyer
Independent electricians and small electrical service businesses

## Site Type
One-page lead generation

## Goal of the Website
Generate quote requests and service enquiries

## Pages Required
- Home

## Required Sections
- Hero
- Services
- Why Choose Us
- Service Areas
- Testimonials
- CTA
- Contact

## Optional Sections
- FAQ
- Emergency Callout
- Project Gallery

## Visual Direction
Clean, dependable, modern trade-professional

## Content Direction
Clear, trustworthy, action-oriented

## Functional Requirements
- Click-to-call actions
- Quote/contact form
- Service area and emergency banner sections

## Technical Constraints
- Static HTML/CSS/JS only
- Minimal JS for mobile nav only
- Optimised for mobile-first local traffic

## Delivery Format
Downloadable static template pack with organised source files and buyer-facing documentation.

## Notes / Special Instructions
This template should feel immediately relevant to its niche while still following the shared production system and reusable section structure.

## Conversion Strategy
Problem awareness → quote request

## Primary User Action
Request a quote

## Secondary Actions
- View services
- Check service areas
- Call now

## Trust Mechanisms
- Licensing and certifications
- Reviews
- Response-time promise

## Content Priority Order
1. Contact CTA
2. Core services
3. Trust and credentials
