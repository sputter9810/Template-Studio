# Project Brief

## Template / Product Name
Wedding Planner Template

## Niche / Industry
Events — Wedding Planner

## Target Customer / Buyer
Wedding planners, coordinators, and boutique event studios

## Site Type
One-page lead generation / enquiry site

## Goal of the Website
Generate wedding enquiries and consultation bookings

## Pages Required
- Home

## Required Sections
- Hero
- Services
- Process
- Portfolio Highlights
- Testimonials
- CTA
- Contact

## Optional Sections
- FAQ
- About
- Venue Partners

## Visual Direction
Elegant, romantic, refined, premium

## Content Direction
Warm, reassuring, polished

## Functional Requirements
- Enquiry/contact form
- Portfolio highlight grid
- Process section with step cards

## Technical Constraints
- Static HTML/CSS/JS only
- Lightweight interactions only
- Responsive image and form handling

## Delivery Format
Downloadable static template pack with organised source files and buyer-facing documentation.

## Notes / Special Instructions
This template should feel immediately relevant to its niche while still following the shared production system and reusable section structure.

## Conversion Strategy
Romantic interest → consultation enquiry

## Primary User Action
Enquire about a date

## Secondary Actions
- View services
- See portfolio highlights
- Read testimonials

## Trust Mechanisms
- Portfolio imagery
- Client testimonials
- Clear process

## Content Priority Order
1. Emotional positioning
2. Services and process
3. Trust and portfolio
