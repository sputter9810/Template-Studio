# Project Brief

## Template / Product Name
Plumber Template

## Niche / Industry
Trade Services — Plumbing

## Target Customer / Buyer
Plumbers and small plumbing businesses

## Site Type
One-page lead generation

## Goal of the Website
Generate service calls and quote enquiries

## Pages Required
- Home

## Required Sections
- Hero
- Services
- Emergency Banner
- Why Choose Us
- Testimonials
- CTA
- Contact

## Optional Sections
- FAQ
- Service Areas
- Maintenance Plans

## Visual Direction
Practical, trustworthy, clean, local-service focused

## Content Direction
Direct, reassuring, problem-solving

## Functional Requirements
- Click-to-call actions
- Quote/contact form
- Emergency or same-day service callout

## Technical Constraints
- Static HTML/CSS/JS only
- No form backend included
- Mobile-first performance priority

## Delivery Format
Downloadable static template pack with organised source files and buyer-facing documentation.

## Notes / Special Instructions
This template should feel immediately relevant to its niche while still following the shared production system and reusable section structure.

## Conversion Strategy
Urgent need → phone call or quote request

## Primary User Action
Call or request a quote

## Secondary Actions
- View services
- Check emergency availability
- Read reviews

## Trust Mechanisms
- Licensed credentials
- Local reviews
- Emergency response positioning

## Content Priority Order
1. Immediate CTA
2. Service list
3. Trust and locality
