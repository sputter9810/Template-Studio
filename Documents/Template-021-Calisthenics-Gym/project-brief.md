# Project Brief

## Template / Product Name
Calisthenics Gym Template

## Niche / Industry
Calisthenics Gym

## Target Customer / Buyer
Independent calisthenics gyms, movement studios, personal trainers, and hybrid strength spaces

## Site Type
One-page sales site with optional timetable/contact expansion

## Goal of the Website
Drive trial class bookings, intro consultations, and membership enquiries while showing coaching quality and community culture.

## Pages Required
- Home
- Optional timetable anchors
- Optional contact/lead capture section

## Required Sections
- Navigation
- Hero
- Programs
- Coaches
- Results / social proof
- Pricing
- FAQ
- CTA
- Contact
- Footer

## Optional Sections
- Gallery
- Amenities
- Schedule

## Visual Direction
Athletic, stripped-back, confident, urban, high-contrast, with editorial training imagery and strong motion cues.

## Content Direction
Outcome-led copy focused on skill progress, strength, mobility, coaching, and community. CTA tone should feel motivating, direct, and practical.

## Functional Requirements
- Sticky nav with anchors
- Lead form for trial session
- Membership/program cards
- Optional class timetable block

## Technical Constraints
- Static HTML/CSS/JS only
- No framework dependency
- Use placeholder images that suit bodyweight training and rings/bars

## Delivery Format
Product-ready static template pack with docs and preview imagery.

## Notes / Special Instructions
Prioritise progress pathways: beginner to advanced. Make skill-led training feel accessible, not intimidating.

## Conversion Strategy
Free trial, intro consult, or membership lead generation.

## Primary User Action
Book a trial / enquire

## Secondary Actions
- View programs
- See timetable
- Read member outcomes

## Trust Mechanisms
- Coach bios
- Member testimonials
- Facility/equipment highlights
- Skill progression framing

## Content Priority Order
1. Hero + CTA
2. Programs / memberships
3. Results + coaches
