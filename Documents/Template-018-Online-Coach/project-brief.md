# Project Brief

## Template / Product Name
Online Coach Template

## Niche / Industry
Digital Business — Online Coach / Course Creator

## Target Customer / Buyer
Online coaches, mentors, and digital product sellers

## Site Type
One-page conversion landing page

## Goal of the Website
Sell coaching calls, programs, or digital products

## Pages Required
- Home

## Required Sections
- Hero
- Offer
- Results
- About Coach
- Testimonials
- CTA
- Contact

## Optional Sections
- FAQ
- Pricing
- Lead Magnet

## Visual Direction
Clean, modern, aspirational, authority-led

## Content Direction
Confident, benefit-driven, simple

## Functional Requirements
- Lead capture/contact form
- Offer section with CTA anchors
- Simple FAQ toggles if used

## Technical Constraints
- Static HTML/CSS/JS only
- No checkout integration included
- Simple JS only for nav/FAQ

## Delivery Format
Downloadable static template pack with organised source files and buyer-facing documentation.

## Notes / Special Instructions
This template should feel immediately relevant to its niche while still following the shared production system and reusable section structure.

## Conversion Strategy
Lead magnet / discovery call → paid offer

## Primary User Action
Book a call or join the program

## Secondary Actions
- Read offer details
- View testimonials
- Learn about the coach

## Trust Mechanisms
- Testimonials
- Results snapshots
- Authority positioning

## Content Priority Order
1. Offer clarity
2. Results
3. Coach credibility
