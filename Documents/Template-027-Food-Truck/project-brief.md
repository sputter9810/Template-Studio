# Project Brief

## Template / Product Name
Food Truck Template

## Niche / Industry
Food Truck

## Target Customer / Buyer
Food trucks, mobile kitchens, event caterers, and pop-up food brands

## Site Type
One-page mobile hospitality site

## Goal of the Website
Help customers find the truck, view signature menu items, and book catering or event enquiries quickly.

## Pages Required
- Home
- Menu highlights
- Contact / bookings

## Required Sections
- Navigation
- Hero
- Menu highlights
- Locations / schedule
- About
- CTA
- Contact
- Footer

## Optional Sections
- Gallery
- Testimonials
- FAQ
- Hours & location

## Visual Direction
Bold, fun, colourful, appetite-led, mobile-first, and energetic.

## Content Direction
Punchy, playful, flavour-led copy with practical details kept obvious.

## Functional Requirements
- Location/schedule section
- Signature food cards
- Catering CTA
- Quick contact details

## Technical Constraints
- Static only
- No live map or live schedule sync
- Keep info editable for moving schedules

## Delivery Format
Static template pack.

## Notes / Special Instructions
Need immediate clarity on what the truck serves and where to find it.

## Conversion Strategy
Catering/event enquiries and in-person visits.

## Primary User Action
Find the truck

## Secondary Actions
- See the menu
- Book catering
- Check this week’s stops

## Trust Mechanisms
- Popular menu items
- Event/catering proof
- Simple schedule
- Social proof

## Content Priority Order
1. Hero + menu
2. Location schedule
3. CTA + contact
