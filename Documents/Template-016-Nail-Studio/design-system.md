# Design System

## Brand Personality
This template should feel clean, stylish, feminine, polished while remaining polished, modern, and easy for buyers to adapt.

## Colour System
### Primary
Soft blush or mauve

### Secondary
Warm cream or sand

### Accent
Rose gold or berry accent

### Neutrals
Light neutrals and charcoal text

### Feedback / Utility Colours
Success, warning, error, and info colours should remain clean, readable, and consistent with the overall palette.

## Typography
### Font Pairing / Style Direction
Elegant display headings + modern sans-serif body

### Type Scale
- H1: Large, high-impact headline suitable for above-the-fold conversion
- H2: Strong section title with clear visual hierarchy
- H3: Compact subsection or card title
- Body: Comfortable paragraph size for web reading
- Small: Useful for support copy, labels, and meta text

## Spacing System
Use a consistent spacing scale with generous section padding, strong content rhythm, and tighter spacing inside cards or lists.

## Radius / Shadows / Borders
Use subtle radius and restrained shadow/border treatment consistent with the niche. Avoid overdecorating unless the niche specifically benefits from it.

## Layout System
Use a centred container, one-page flow, flexible card grids, and stacked mobile layouts. Prioritise clear CTA placement and scannable section groupings.

## Breakpoints
- Mobile: 480px
- Tablet: 768px
- Desktop: 1024px
- Large Desktop: 1280px+

## Components
### Buttons
Primary buttons should support the main conversion goal: Book your next appointment. Secondary buttons can support exploration such as pricing, schedule, or gallery.

### Cards
Cards should be reusable for services, pricing, and visual gallery, testimonials, team items, process steps, and FAQs where needed.

### Forms
Forms should be simple, high-contrast, and easy to complete on mobile. Labels should be visible and not placeholder-only.

### Navigation
Use a clean sticky or semi-sticky navigation with anchor links to the major sections.

### Badges / Tags
Badges can be used sparingly for highlights such as featured services, intro offers, credentials, or category labels.

## Imagery / Placeholder Rules
Use realistic placeholder imagery that fits the niche. Prioritise photos that make the business feel believable and conversion-ready.

## Interaction Style
Keep transitions light and purposeful. Hover, focus, and reveal states should improve clarity rather than add novelty.

## Accessibility Rules
- Maintain strong contrast for text and interactive elements
- Ensure all buttons and links have visible focus states
- Keep tap targets comfortable on mobile
- Use clear hierarchy and readable line lengths

## Notes
The design should sell the niche quickly, but the structure should still feel reusable for close variants.
