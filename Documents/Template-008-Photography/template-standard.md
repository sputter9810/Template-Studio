# Template Standards

## Required Sections
Every template must include:

- Navigation
- Hero
- Core Offer (services/menu/programs)
- About / Trust
- Testimonials
- CTA
- Contact
- Footer

## Optional Sections
Optional sections:

- Process
- FAQ
- Gallery
- Pricing
- Team
- Portfolio
- Trust Strip

## Naming Rules
Rules:

- Base classes are generic: `.card`, `.section`, `.btn`
- Variants are modifiers: `.service-card`, `.hero-with-stats`
- No niche-specific base classes

## Folder Structure

src/
  index.html
  css/
    style.css
    base/
    layout/
    components/
    sections/
  js/
    main.js
  images/
docs/
  customisation-guide.md
  README.md

## Done
A template is complete when:

- all sections are responsive
- all CTAs are wired (anchors)
- no unused CSS exists
- consistent spacing system used
- README is written
