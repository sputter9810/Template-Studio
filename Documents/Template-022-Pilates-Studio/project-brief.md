# Project Brief

## Template / Product Name
Pilates Studio Template

## Niche / Industry
Pilates Studio

## Target Customer / Buyer
Boutique pilates studios, reformer studios, and movement/wellness spaces

## Site Type
One-page studio website

## Goal of the Website
Generate class bookings and intro package enquiries while communicating calm professionalism and results.

## Pages Required
- Home
- Optional class schedule area
- Contact / booking

## Required Sections
- Navigation
- Hero
- Class styles
- About
- Schedule
- Pricing
- Testimonials
- CTA
- Contact
- Footer

## Optional Sections
- Amenities
- FAQ
- Team
- Gallery

## Visual Direction
Soft, refined, calm, airy, premium wellness aesthetic with generous spacing and clean photography.

## Content Direction
Warm, reassuring, polished copy focused on mobility, strength, posture, and mindful movement.

## Functional Requirements
- Anchor navigation
- Class timetable section
- Pricing/package cards
- Booking CTA to external platform

## Technical Constraints
- Static front-end only
- No real booking integration in core
- Use neutral wellness placeholders

## Delivery Format
Static product template with clear documentation.

## Notes / Special Instructions
Keep copy approachable for beginners while still feeling premium.

## Conversion Strategy
Intro pack sales and booking enquiries.

## Primary User Action
Book your first class

## Secondary Actions
- View schedule
- See pricing
- Meet the instructors

## Trust Mechanisms
- Instructor bios
- Class levels clearly explained
- Studio inclusions
- Testimonials

## Content Priority Order
1. Hero + booking CTA
2. Schedule + pricing
3. About + testimonials
