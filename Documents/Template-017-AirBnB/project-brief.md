# Project Brief

## Template / Product Name
Airbnb / Holiday Rental Template

## Niche / Industry
Hospitality — Short Stay / Holiday Rental

## Target Customer / Buyer
Airbnb hosts, holiday rental owners, and boutique stay operators

## Site Type
One-page direct booking / enquiry site

## Goal of the Website
Increase direct stay enquiries and bookings

## Pages Required
- Home

## Required Sections
- Hero
- Gallery
- Amenities
- Stay Details
- Testimonials
- CTA
- Contact

## Optional Sections
- Local Guide
- FAQ
- Rates

## Visual Direction
Warm, inviting, travel-inspired, premium but approachable

## Content Direction
Descriptive, escapist, conversion-friendly

## Functional Requirements
- Booking enquiry form
- Image gallery
- Availability and amenity highlight blocks

## Technical Constraints
- Static HTML/CSS/JS only
- No live booking engine included
- Fast performance with optimised placeholders

## Delivery Format
Downloadable static template pack with organised source files and buyer-facing documentation.

## Notes / Special Instructions
This template should feel immediately relevant to its niche while still following the shared production system and reusable section structure.

## Conversion Strategy
Property interest → booking enquiry

## Primary User Action
Check availability / book stay

## Secondary Actions
- View amenities
- See gallery
- Read local highlights

## Trust Mechanisms
- Property photography
- Guest reviews
- Amenity list and location appeal

## Content Priority Order
1. Hero and stay offer
2. Gallery / amenities
3. Reviews and local appeal
