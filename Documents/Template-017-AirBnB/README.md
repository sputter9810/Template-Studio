# Airbnb / Holiday Rental Template

## Summary
This folder contains the filled strategy and standards files for a Hospitality — Short Stay / Holiday Rental website template.

## Goal
The template is designed to help airbnb hosts, holiday rental owners, and boutique stay operators turn visitors into customers through a focused one-page conversion flow.

## Included Files
- project-brief.md
- design-system.md
- content-guide.md
- production-system.md
- technical-standards.md
- template-standard.md
- qa-checklist.md
- product-checklist.md

## Primary Conversion Goal
Check availability / book stay
