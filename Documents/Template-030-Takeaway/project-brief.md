# Project Brief

## Template / Product Name
Takeaway Restaurant Template

## Niche / Industry
Takeaway

## Target Customer / Buyer
Takeaway restaurants, quick-service venues, local delivery-first kitchens, and casual eateries

## Site Type
One-page conversion site

## Goal of the Website
Get customers from landing to ordering fast by making menu, hours, and ordering actions obvious.

## Pages Required
- Home
- Menu highlights
- Order/contact

## Required Sections
- Navigation
- Hero
- Menu highlights
- Trust or reviews
- CTA
- Contact
- Footer

## Optional Sections
- Hours & location
- FAQ
- Gallery
- Signature offers

## Visual Direction
Clean, bold, appetising, and efficient with strong action hierarchy.

## Content Direction
Fast-scanning copy built around menu highlights, ordering convenience, and local trust.

## Functional Requirements
- Order CTA buttons
- Menu cards
- Hours/location block
- Delivery or pickup notes

## Technical Constraints
- Static only
- No live ordering integration in core
- Keep mobile performance high and copy compact

## Delivery Format
Static template pack.

## Notes / Special Instructions
Should suit burgers, pizza, sushi, chicken, bowls, or other casual food concepts.

## Conversion Strategy
Orders, pickup visits, and delivery CTA clicks.

## Primary User Action
Order now

## Secondary Actions
- See the menu
- Check opening hours
- Get directions

## Trust Mechanisms
- Review snippets
- Fast service cues
- Local popularity
- Popular item highlights

## Content Priority Order
1. Hero + order CTA
2. Menu highlights
3. Hours/location/contact
