# Project Brief

## Template / Product Name
CrossFit Gym Template

## Niche / Industry
Fitness — CrossFit

## Target Customer / Buyer
CrossFit gym owners and functional fitness coaches

## Site Type
One-page lead generation

## Goal of the Website
Convert visitors into trial bookings

## Pages Required
- Home

## Required Sections
- Hero
- Programs
- Schedule
- Coaches
- Testimonials
- CTA
- Contact

## Optional Sections
- FAQ
- Pricing
- Gallery

## Visual Direction
Dark, bold, high-energy

## Content Direction
Motivational, results-driven

## Functional Requirements
- Sticky nav with anchor links
- Trial booking/contact form
- Mobile-friendly class schedule block

## Technical Constraints
- Static HTML/CSS/JS only
- No external framework dependency
- Fast load with optimised placeholders

## Delivery Format
Downloadable static template pack with organised source files and buyer-facing documentation.

## Notes / Special Instructions
This template should feel immediately relevant to its niche while still following the shared production system and reusable section structure.

## Conversion Strategy
Free trial → membership signup

## Primary User Action
Book a free trial

## Secondary Actions
- View class schedule
- Meet coaches
- Read testimonials

## Trust Mechanisms
- Transformation results
- Testimonials
- Class stats

## Content Priority Order
1. Offer (trial)
2. Results
3. Community
