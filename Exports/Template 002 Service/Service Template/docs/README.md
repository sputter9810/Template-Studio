# Northline Trade Co. — Tradies / Services Website Template

A polished one-page website template built for tradies and service businesses that want a clean, trustworthy online presence and more enquiries.

## What this template is for

This template is designed for small local service businesses such as:
- electricians
- plumbers
- builders
- landscapers
- painters
- handymen
- cleaners
- general contractors

It works especially well for businesses that want a simple brochure-style website with strong calls to action, trust-building sections, and a clear contact form.

## What is included

- responsive one-page homepage
- mobile navigation
- hero section with call-to-action buttons
- trust strip
- services section
- about section
- optional gallery section
- process section
- testimonials section
- FAQ section
- contact section with enquiry form UI
- placeholder SVG images
- modular CSS structure for easier editing
- customisation guide for setup and editing

## Before you use it

You will usually want to:
- replace the business name
- update the colours to your brand
- swap placeholder images for real photos
- edit the service cards and contact details
- connect the form to your preferred backend or form service

## Technical notes

This is a static template built with:
- HTML
- CSS
- JavaScript

No build tools or frameworks are required. You can open the homepage directly in a browser or upload the `src` folder to your preferred static hosting platform.
