# Harbour & Key — Real Estate Website Template

Harbour & Key is a premium one-page static website template designed for real-estate agents, boutique agencies, property marketers, and appraisal-focused landing pages.

## What this template is for
This template is built to help property businesses present listings clearly, establish trust quickly, and drive enquiries through a polished lead-generation layout.

It works well for:
- solo agents
- boutique real-estate agencies
- leasing teams
- property marketers
- appraisal and vendor campaign pages
- buyer enquiry or suburb-profile style sites

## What is included
The template includes a complete one-page structure with:
- sticky header navigation
- hero section with trust stats
- trust strip
- featured listings section
- about / credibility section
- process section
- gallery section
- testimonials
- FAQ
- call-to-action banner
- contact form
- footer

## Key product traits
- clean, modern real-estate aesthetic
- responsive layout for mobile through desktop
- modular CSS structure for easier editing
- placeholder visuals ready for image replacement
- static contact form ready to connect to your preferred form handler
- suitable for Netlify, Vercel, GitHub Pages, or similar static hosting

## Notes
This is a static front-end template. It does not include a CMS, live property feed, database, or backend form processing out of the box.
