# Forge Fitness — Static Gym Website Template

Forge Fitness is a modern one-page website template designed for gyms, fitness studios, strength and conditioning spaces, and personal training businesses.

## What this template is for
Use this template to launch a professional marketing site for a fitness business that wants to:
- showcase training programs
- build trust quickly
- present the facility and brand clearly
- collect leads through an enquiry or trial form
- provide a polished mobile-friendly first impression

## Included pages and files
- 1 responsive homepage
- modular CSS architecture
- lightweight JavaScript for mobile navigation and demo form handling
- customisation guide

## Best suited for
- gyms
- boutique fitness studios
- personal trainers
- strength and conditioning brands
- semi-private training businesses
- group training facilities

## Key sections included
- sticky navigation
- hero section with calls to action
- trust strip
- programs/services cards
- about section
- gallery/facility section
- simple onboarding process section
- testimonials
- FAQ
- CTA banner
- contact section with form
- footer

## Tech stack
- HTML
- CSS
- JavaScript

No build tools or frameworks are required.

## Notes
- Images are represented with styled placeholder panels for easy reuse and faster customisation.
- The form is a static demo and should be connected to a live backend or form handler before production use.
- The template is built to be easy to rebrand for different fitness businesses.


## Theme selector
This template now includes a built-in theme selector panel with five colour schemes: Graphite, Neon Lime, Crimson, Deep Blue, White / Black.

To change the default theme, update the `data-theme` value on the `<body>` tag and the `defaultTheme` value in `src/assets/js/main.js`.
