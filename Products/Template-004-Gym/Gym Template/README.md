# Forge Fitness — Gym Website Template

A clean, modern one-page static website template for gyms, fitness studios, and personal training businesses.

## Included

- `index.html`
- `assets/css/style.css`
- `assets/js/main.js`
- `assets/images/`
- `docs/customisation-guide.md`

## Features

- One-page lead-generation layout
- Mobile responsive navigation
- Hero, trust strip, programs, about, gallery, process, testimonials, FAQ, CTA, and contact sections
- Static demo contact form
- Placeholder media panels for easy visual replacement

## Built With

- HTML
- CSS
- JavaScript

## Customise

Update the following to adapt the template:

- brand name in the header and footer
- colours in `assets/css/style.css`
- section copy in `index.html`
- contact details and CTA text
- placeholder panels with real photos or media

## Notes

This is a static template. The contact form is presentation-ready but needs to be connected to a backend or form service for live use.
