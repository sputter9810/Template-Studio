# Ever After Atelier | Wedding Planner Website Template

Ever After Atelier is a static one-page website template created for wedding planners, coordinators, and boutique event studios. It is designed to help buyers launch a polished enquiry-driven website quickly, with sections focused on services, process, portfolio highlights, trust signals, and consultation bookings.

## What this template is for

This template is intended for:

- wedding planners
- wedding coordinators
- boutique event studios
- luxury and premium celebration planners
- destination wedding planners
- styling-led planning businesses

It works especially well for service-based wedding businesses that want a refined landing page rather than a large multi-page site.

## What is included

Inside this product you will find:

- a complete static homepage
- modular CSS files organised into base, layout, components, and sections
- a small JavaScript file for the mobile navigation toggle and theme selector
- a customer customisation guide
- elegant placeholder copy and visual panels that are easy to adapt

## Notes for buyers

- This is a static HTML, CSS, and JavaScript template.
- The contact form is a front-end demo form and will need a form handler or backend to send live submissions.
- Placeholder panels are used instead of real wedding imagery so the template can be reused and branded easily.
- The structure is modular to make editing and scaling simpler for non-technical buyers.

## Tech used

- HTML5
- CSS3
- vanilla JavaScript

## Theme selector

This template includes a built-in theme selector panel with five colour schemes: Champagne, Soft Blush, Sage Garden, Black Tie, and Pearl.

To change the default theme, update the `data-theme` value on the `<body>` tag and the `defaultTheme` value in `src/assets/js/main.js`.
