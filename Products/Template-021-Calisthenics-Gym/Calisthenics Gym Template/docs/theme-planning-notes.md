# Theme Planning Notes

## Default direction
- Dark, confident, movement-first
- Calisthenics-specific positioning with beginner-friendly credibility
- Focus on trial bookings, skill progression, and coach-led structure

## Theme variants
- Graphite Lime: charcoal base with an electric lime accent for the flagship athletic look
- Urban Teal: darker studio palette with a fresh teal highlight
- Steel Mint: cooler steel-led palette with cleaner mint contrast
- Night Heat: warmer orange-led option for a more energetic training feel
- Chalk Studio: light mode variant for buyers who want a brighter presentation

## Typography direction
- Barlow Condensed for strong display headings
- Inter for readable body copy and UI text
