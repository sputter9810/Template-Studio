# Northstar Agency

A clean, modern one-page static website template for agencies.

## Best suited for
- creative agencies
- digital marketing agencies
- branding studios
- consultants and service businesses using an agency-style offer

## Included sections
- sticky navigation
- hero with stats
- trust strip
- services
- about / trust
- process
- portfolio gallery
- testimonials
- FAQ
- CTA banner
- contact form
- footer

## Tech
- HTML
- CSS
- JavaScript

## Folder structure
```text
index.html
assets/
  css/
    style.css
  js/
    main.js
  images/
docs/
  customisation-guide.md
README.md
```

## Customisation
Update `index.html` to change the brand name, copy, service cards, testimonials, contact details, and CTA labels.

Update `assets/css/style.css` to change colours, spacing, typography, and container width.

## Form handling
The included form is a static demo form. Connect it to your preferred backend, email workflow, or no-code form service for live submissions.

## Deployment
You can deploy this template to Netlify, Vercel, GitHub Pages, or any standard static hosting provider.
