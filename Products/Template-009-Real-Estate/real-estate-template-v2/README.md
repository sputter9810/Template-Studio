# Harbour & Key — Real Estate Template

A clean one-page static website template for real-estate agents, boutique agencies, and property marketers.

## Who it is for
- solo agents
- boutique real-estate agencies
- property marketers
- appraisal and vendor lead pages

## Sections included
- Navigation
- Hero
- Trust Strip
- Featured Listings / Core Offer
- About / Trust
- Process
- Gallery
- Testimonials
- FAQ
- CTA Banner
- Contact
- Footer

## Tech stack
- HTML
- CSS
- JavaScript

## Folder structure
```text
src/
  index.html
  css/
    style.css
    base/
    layout/
    components/
    sections/
  js/
    main.js
  images/
docs/
  customisation-guide.md
README.md
```

## Customisation basics
1. Replace the brand name and initials in `src/index.html`.
2. Update colours in `src/css/base/variables.css`.
3. Swap placeholder copy and listing details with your own content.
4. Connect the contact form to your preferred handler.
5. Add your own imagery in `src/images/` and update the relevant sections.

## Deployment
This template can be deployed to Netlify, Vercel, GitHub Pages, or any static host.
