# Harbour & Key — Real Estate Template

Harbour & Key is a clean, modern one-page website template built for real-estate businesses, boutique agencies, buyer advocates, and independent agents.

## Who it's for

This template is suited to:

- boutique real-estate agencies
- solo real-estate agents
- buyer advocates
- property consultants
- small businesses that want a polished lead-generation site

## Sections included

- Sticky navigation
- Hero with stats and CTAs
- Trust strip
- Featured listings / core offer
- About / trust section
- Process section
- Gallery
- Testimonials
- FAQ
- CTA banner
- Contact section with form
- Footer

## Tech stack

## How to use

1. Open `src/index.html` in your browser.
2. Replace placeholder copy with your own business content.
3. Update colours in `src/css/base/variables.css`.
4. Replace placeholder panels with real images or media.
5. Connect the contact form to your preferred static form service or backend.

## How to deploy

This template can be deployed as a static site on platforms such as Netlify, Vercel, GitHub Pages, or any standard web host.

Basic deployment flow:

1. Keep the `src` folder contents together.
2. Upload the site files to your chosen platform.
3. Set the publish directory to `src` if your platform asks for one.
4. Test links, form behaviour, and mobile navigation after deployment.

## Notes

- Images are intentionally represented as neutral placeholder panels.
- The form is a static demo form by default.
- All CTA buttons use anchor links for easy one-page navigation.
