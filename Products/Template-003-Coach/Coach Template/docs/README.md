# Momentum Coach

Momentum Coach is a clean, modern one-page website template built for trainers, coaches, and personal trainers. It is designed to help service-based fitness brands present their offer clearly, build trust quickly, and encourage enquiries or bookings.

## What this template is for

This template suits businesses and personal brands such as:

- personal trainers
- online fitness coaches
- strength and conditioning coaches
- nutrition coaches
- wellness and performance coaches

It works best for businesses that want a focused landing-page style site with a strong call to action.

## What is included

Inside the product you will find:

- a complete one-page website in HTML, CSS, and JavaScript
- modular CSS files organised into base, layout, components, and sections
- a responsive mobile navigation
- sections for hero, trust strip, services, about, process, testimonials, FAQ, CTA, contact, and footer
- a customisation guide for updating branding, colours, and content

## Notes

This is a static template. The contact form is presentational by default and will need to be connected to your preferred form handling service, booking flow, or backend.

Images shown in the design are represented with styled content panels, so you can replace them with your own photography or brand visuals easily.


## Theme selector
This template now includes a built-in theme selector panel with five colour schemes: Red / Black, Electric Blue, Emerald, Sand, Purple / Black.

To change the default theme, update the `data-theme` value on the `<body>` tag and the `defaultTheme` value in `src/assets/js/main.js`.
