# Momentum Coach

A static one-page website template for trainers, coaches, and personal trainers.

## Folder Structure

```text
index.html
assets/
  css/
    style.css
  images/
  js/
    main.js
docs/
  customisation-guide.md
README.md
```

## Features

- One-page lead generation layout
- Sticky navigation with mobile toggle
- Hero, services, about, process, testimonials, FAQ, CTA, contact, and footer
- Modern warm coaching-focused styling
- Static contact form ready to connect to a live handler
