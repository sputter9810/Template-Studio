# Stillspace - Static Yoga Studio Website Template

Stillspace is a calm one-page static website template made for yoga studios, yoga teachers, and boutique wellness operators that want more intro class bookings.

## What this template is for
Use this template to:
- promote an intro pass clearly
- show class styles in a friendly, beginner-ready way
- display a mobile-friendly timetable
- introduce teachers and studio trust signals
- guide visitors toward an intro class enquiry

## Included sections
- sticky navigation
- hero section with CTA buttons and stats
- trust strip
- class styles section
- schedule section
- about section
- teachers section
- testimonials
- FAQ
- CTA banner
- contact section with form
- footer

## Theme selector
This template includes a built-in theme selector panel with five color schemes: Sage, Clay, Ocean, Dusk, and Linen.
