# North Roast - Static Coffee Roaster Website Template

North Roast is a premium one-page static website template made for independent coffee roasters, specialty coffee brands, and cafe-roaster hybrids that want more bean sales, subscriptions, wholesale enquiries, and cafe visits.

## What this template is for
Use this template to:
- showcase featured coffees with realistic tasting notes and product-first copy
- lead visitors toward a coffee purchase, subscription signup, or wholesale enquiry
- reinforce trust with sourcing cues, roasting philosophy, process, and testimonials
- make retail hours, location, and pickup details easy to find on mobile
- support direct-to-consumer and trade conversion paths without requiring a full ecommerce build

## Included sections
- sticky navigation
- hero section with CTA buttons and trust stats
- trust strip
- featured coffees section
- subscriptions and offers section
- brand story / roasting philosophy section
- roasting process section
- testimonials
- visit section with hours and location
- FAQ
- CTA banner
- order and wholesale section with form
- footer
