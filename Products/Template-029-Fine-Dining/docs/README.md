# Velour - Static Fine Dining Website Template

Velour is a moody one-page static website template made for fine dining restaurants, chef-led concepts, degustation venues, and premium hospitality brands that want more reservations and private dining enquiries.

## What this template is for
Use this template to:
- showcase a premium dining experience with seasonal menu storytelling and strong reservation cues
- encourage guests to reserve a table, enquire about chef's table seating, or ask about private dining
- reinforce trust with chef story, polished testimonials, and clear service details
- make hours, address, and reservation notes easy to find on mobile
- support tasting-menu restaurants, elevated a la carte concepts, and premium hospitality brands without needing a live booking engine in the core pack

## Included sections
- sticky navigation
- hero section with CTA buttons and trust stats
- trust strip
- signature experience section
- reservations and offers section
- chef story section
- experience highlights section
- testimonials
- hours and location section
- FAQ
- CTA banner
- reservations and contact section with form
- footer
