# Solstice Sanctuary - Static Day Spa Website Template

Solstice Sanctuary is a premium one-page static website template made for boutique day spas, luxury wellness venues, and retreat-style treatment brands that want more package bookings, ritual enquiries, and gift voucher interest.

## What this template is for
Use this template to:
- showcase massage, facial, and body ritual offers with calm premium copy
- lead guests toward a signature package or treatment booking
- reinforce trust with therapist bios, testimonials, and arrival details
- make hours, location, and reservation paths easy to find on mobile
- support gift voucher and occasion-based enquiries without extra pages

## Included sections
- sticky navigation
- hero section with CTA buttons and trust stats
- trust strip
- treatments section
- packages section
- spa experience section
- therapist section
- testimonials
- visit section with hours and location
- FAQ
- CTA banner
- reservation section with form
- footer
