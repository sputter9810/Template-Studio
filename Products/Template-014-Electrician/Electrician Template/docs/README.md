# North Circuit Electrical - Electrician Website Template

A polished one-page website template built for electricians who want a clean, dependable online presence and more local quote requests.

## What this template is for

This template is designed for:
- independent electricians
- electrical contractors
- small electrical service teams
- residential and commercial maintenance providers

It is especially suited to businesses that want a mobile-first lead generation page with strong quote CTAs, emergency callout messaging, service-area coverage, and trust-building proof.

## What is included

- responsive one-page homepage
- mobile navigation
- hero section with quote and click-to-call actions
- trust strip
- emergency callout banner
- services section
- why choose us section
- service areas section
- recent work gallery
- testimonials section
- FAQ section
- contact section with quote form UI
- placeholder SVG images
- modular CSS structure for easier editing
- customisation guide for setup and editing

## Before you use it

You will usually want to:
- replace the business name and contact details
- update the suburbs and service area list
- swap placeholder images for real project photos
- customise the service cards to match the business
- connect the form to your preferred backend or form service

## Technical notes

This is a static template built with:
- HTML
- CSS
- JavaScript

No build tools or frameworks are required. You can open the homepage directly in a browser or upload the `src` folder to your preferred static hosting platform.

## Theme selector
This template includes a built-in theme selector panel with five electrician-friendly colour schemes: Midnight, Signal Amber, Graphite, Electric Blue, and Blackout.

To change the default theme, update the `data-theme` value on the `<body>` tag and the `defaultTheme` value in `src/assets/js/main.js`.
