# Quickdish - Static Takeaway Website Template

Quickdish is a bold one-page static website template made for takeaway restaurants, quick-service venues, delivery-first kitchens, and casual food brands that want more orders, easier menu scanning, and clearer pickup or delivery actions.

## What this template is for
Use this template to:
- showcase bestsellers, combo meals, and local favourites with realistic appetite-led placeholder copy
- help visitors order fast by repeating strong order CTAs throughout the page
- reinforce trust with review snippets, speed cues, and clear service details
- make hours, pickup notes, and location information easy to find on mobile
- stay easy to customise for burgers, pizza, sushi, bowls, wraps, fried chicken, noodles, and similar takeaway concepts

## Included sections
- sticky navigation
- hero section with CTA buttons and trust stats
- trust strip
- menu highlights section
- offers and order options section
- takeaway story section
- service-fit section for speed, delivery, and group orders
- testimonials
- hours and location section
- FAQ
- CTA banner
- order and contact section with form
- footer
