# Northline - Static Barber Shop Website Template

Northline is a bold one-page static website template made for barbershops and independent barbers who want more appointment bookings and stronger local trust.

## What this template is for
Use this template to:
- promote the core service offer clearly
- show service pricing without friction
- introduce the barber team and shop vibe
- build trust with testimonials and gallery content
- guide visitors toward a booking enquiry

## Included sections
- sticky navigation
- hero section with CTA buttons and stats
- trust strip
- services section
- pricing section
- about section
- team section
- gallery
- testimonials
- FAQ
- CTA banner
- contact section with form
- footer

## Theme selector
This template includes a built-in theme selector panel with five color schemes: Charcoal, Walnut, Oxblood, Steel, and Cream.
