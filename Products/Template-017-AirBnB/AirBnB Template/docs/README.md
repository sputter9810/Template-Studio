# Solstice Stays - Holiday Rental Website Template

Solstice Stays is a warm, premium one-page static website template designed for Airbnb hosts, holiday-rental owners, boutique stay operators, and direct booking landing pages.

## What this template is for
This template helps short-stay businesses present a property clearly, build trust fast, and turn visitor interest into booking enquiries through a polished one-page layout.

It works well for:
- Airbnb hosts
- holiday-rental owners
- boutique villas and cabins
- coastal homes and retreats
- serviced apartments
- short-stay property managers

## What is included
The template includes a complete one-page structure with:
- sticky header navigation
- hero section with trust stats
- trust strip
- amenities and booking highlights section
- stay details / trust section
- local guide section
- gallery section
- guest reviews
- FAQ
- call-to-action banner
- contact form
- footer

## Key product traits
- warm, travel-inspired holiday-rental aesthetic
- responsive layout for mobile through desktop
- modular CSS structure for easier editing
- placeholder visuals ready for image replacement
- static enquiry form ready to connect to your preferred form handler
- suitable for Netlify, Vercel, GitHub Pages, or similar static hosting

## Notes
This is a static front-end template. It does not include a live booking engine, CMS, database, or backend form processing out of the box.

## Theme selector
This template includes a built-in theme selector panel with five colour schemes: Olive Coast, Terracotta Dusk, Ocean Retreat, Sandstone, and Night Cabin.

To change the default theme, update the `data-theme` value on the `<body>` tag and the `defaultTheme` value in `src/assets/js/main.js`.
