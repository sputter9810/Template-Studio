# Northlane Coach - Online Coach Template

Northlane Coach is a premium one-page static website template for online coaches, mentors, consultants with a signature method, and digital product sellers who want more qualified leads, booked calls, and program applications.

## What this template is for
This template is designed for businesses that need to:
- clarify the core offer immediately above the fold
- build trust with results, testimonials, and authority-led positioning
- promote a lead magnet, strategy call, or flagship program
- present pricing in a clean and conversion-friendly way
- launch a polished static website without a build process

## Included pages and content
This product includes one homepage with the following sections:
- sticky navigation
- hero section with CTA buttons and stats
- trust strip
- core offer section
- results section
- about coach section
- lead magnet section
- testimonials section
- pricing section
- FAQ section
- CTA banner
- contact section with form
- footer

## File overview
- `src/index.html` - the main website file
- `src/assets/css/style.css` - the CSS entry point that imports all modular CSS files
- `src/assets/css/base/` - global resets, variables, typography, and utility helpers
- `src/assets/css/layout/` - shared layout rules such as container sizing
- `src/assets/css/components/` - reusable UI pieces like buttons, cards, navigation, forms, proof blocks, and footer elements
- `src/assets/css/sections/` - section-specific styles for hero, offer, results, about, lead magnet, pricing, FAQ, CTA, and contact
- `src/assets/js/main.js` - mobile navigation, static demo-form behavior, and theme selector support
- `src/assets/images/` - place replacement images here
- `docs/customisation-guide.md` - editing guidance for developers or implementers

## Editing level
This is a static HTML/CSS/JS template. It is best suited to buyers who are comfortable editing text, swapping images, and making light HTML/CSS changes, or who are working with a developer.

## Form behavior
The contact form is included as a styled static form. It does not submit anywhere until it is connected to a backend, booking flow, CRM, or form service.

## Hosting
You can deploy this template to most static hosting providers, including Netlify, Vercel, GitHub Pages, and similar platforms.

## Theme selector
This template includes a built-in theme selector panel with five color schemes: Plum Authority, Forest Ledger, Navy Lift, Coral Momentum, and Soft Light.

To change the default theme, update the `data-theme` value on the `<body>` tag and the `defaultTheme` value in `src/assets/js/main.js`.
