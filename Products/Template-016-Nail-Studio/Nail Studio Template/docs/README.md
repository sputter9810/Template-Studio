# Gloss Studio - Customer Overview

Gloss Studio is a static one-page website template for nail salons, boutique nail studios, and independent nail technicians.

## What this template is for
This template is designed to help beauty businesses present their services clearly, show visual proof of their work, and guide visitors toward booking an appointment.

## What is included
- A polished one-page homepage
- Nail studio hero section
- Services section
- Pricing section
- About section
- Gallery section
- Testimonials section
- FAQ section
- Contact section with styled booking form
- Mobile navigation
- Modular CSS file structure for easy edits

## Best suited for
- Nail salons
- Independent nail artists
- BIAB and gel specialists
- Boutique beauty studios
- Small teams wanting a booking-focused site

## Technical format
This is a static HTML, CSS, and JavaScript template. It does not require a framework to run.

## Important note
The included contact form is pre-styled but does not send submissions on its own. A form service or backend connection is needed for live bookings.

## Theme selector
This template includes a built-in theme selector panel with five colour schemes: Blush, Berry, Champagne, Sage, and After Hours.

To change the default theme, update the `data-theme` value on the `<body>` tag and the `defaultTheme` value in `src/assets/js/main.js`.
