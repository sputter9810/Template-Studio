# Lens & Light — Photographer Website Template

Lens & Light is a static one-page website template created for photographers and photography-led creative businesses. It is designed to help a buyer launch a polished online presence quickly, with sections focused on showcasing services, portfolio work, trust signals, and enquiries.

## What this template is for

This template is intended for:

- portrait photographers
- wedding photographers
- lifestyle photographers
- event photographers
- studio photographers
- personal brand and editorial photographers

It works especially well for service-based photography businesses that want a clean landing page rather than a large multi-page website.

## What is included

Inside this product you will find:

- a complete static homepage
- modular CSS files organised into base, layout, components, and sections
- a small JavaScript file for the mobile navigation toggle
- a customer customisation guide
- a neutral visual style that is easy to rebrand

## Sections included in the page

- sticky navigation
- hero

- services
- about
- portfolio gallery
- process
- testimonials

## Notes for buyers

- This is a static HTML, CSS, and JavaScript template.
- The contact form is a front-end demo form and will need a form handler or backend to send live submissions.
- Placeholder panels are used instead of real photos so the template can be reused and branded easily.
- The structure is modular to make editing and scaling simpler.
