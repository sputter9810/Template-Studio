# Lens & Light — Photographer Template

Lens & Light is a clean, modern static website template designed for photographers who want to showcase services, highlight portfolio work, and generate enquiries.

## Who it is for

This template is suited to:

- portrait photographers
- wedding photographers
- lifestyle photographers
- event photographers
- personal brand and commercial photographers

## Sections included

- Sticky navigation
- Hero
- Trust strip
- Services
- About
- Gallery
- Process
- Testimonials
- CTA banner
- Contact form
- Footer

## Tech stack

- HTML
- CSS
- JavaScript

## Folder structure

```text
index.html
assets/
  css/
    style.css
  js/
    main.js
  images/
docs/
  customisation-guide.md
```

## Customisation

Open `index.html` and replace:

- brand name
- headings and body copy
- contact details
- placeholder portfolio labels
- CTA text

Open `assets/css/style.css` to adjust:

- colour palette
- typography scale
- spacing
- border radius
- layout widths

## Images

This template uses neutral placeholder panels instead of real images so it is easy to reuse and rebrand. Replace the placeholder panels with your own photography when preparing the template for a client or marketplace listing.

## Deployment

This is a fully static template and can be deployed to:

- Netlify
- Vercel
- GitHub Pages
- any standard web host

## Notes

The contact form is a static demo form. Connect it to your preferred backend, form handler, or automation service for live submissions.
