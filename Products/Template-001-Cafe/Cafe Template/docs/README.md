# Harbour & Hearth Cafe Template

## What this template is
Harbour & Hearth is a modern one-page website template designed for cafes, brunch venues, bakeries, espresso bars, and boutique hospitality businesses.

## What it is for
This template is made to help you launch a polished online presence quickly. It is ideal for:
- showcasing your venue and brand atmosphere
- highlighting featured menu items
- presenting opening hours and contact details
- encouraging table bookings and general enquiries

## What is included
- responsive homepage layout
- hero section with call-to-action buttons
- featured menu cards
- about section
- gallery section
- visit / process section
- testimonials
- FAQ
- contact section with demo form
- organised CSS structure
- simple JavaScript for mobile navigation

## Folder overview
- `src/` contains the website files
- `docs/customisation-guide.md` explains how to edit the template
- `docs/README.md` is this customer-facing overview

## Editing the template
You can update text, colours, imagery, and section order without changing the overall structure. The codebase is organised to make customisation straightforward.

## Important note
This is a static HTML/CSS/JS template. The form is included as a styled demo and can be connected to your preferred backend, form handler, or booking tool.
