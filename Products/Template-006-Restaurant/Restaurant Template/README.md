# Harvest Table — Restaurant Template

Harvest Table is a clean, modern one-page website template for restaurants, bistros, wine bars, and similar hospitality businesses.

## Who it is for
- Small restaurant owners
- Hospitality venues needing a polished online presence
- Freelancers or agencies building quick brochure-style restaurant sites

## Sections included
- Navigation
- Hero
- Trust strip
- Signature menu
- About
- Gallery
- Testimonials
- FAQ
- CTA
- Contact
- Footer

## Tech stack
- HTML
- CSS
- JavaScript

## Features
- Mobile responsive layout
- Sticky navigation with mobile toggle
- Reusable component-based CSS structure
- Placeholder imagery blocks for easy replacement
- Styled enquiry form ready for backend integration

## How to use
1. Open `src/index.html` in your browser.
2. Replace the placeholder brand, copy, and contact information.
3. Swap placeholder panels with your own restaurant photography.
4. Update colours in `src/css/base/variables.css`.
5. Connect the contact form to your preferred backend or form handler if needed.

## Deployment
This template can be deployed to any static hosting platform, including Netlify, Vercel, GitHub Pages, or a traditional cPanel host.

## File structure
- `src/` — template source files
- `docs/` — customisation and QA notes

## Notes
The form is static by default and does not submit to a live endpoint.
