# Harvest Table Template Overview

Harvest Table is a static one-page website template for restaurants, bistros, cafes, wine bars, and other hospitality venues that want a polished online presence without the overhead of a full custom build.

## What this template is for
- Presenting a venue in a clean, modern way
- Showcasing signature dishes and experience-led branding
- Building trust through reviews, gallery content, and venue information
- Capturing enquiries for bookings, private dining, or events
- Giving freelancers, studios, or business owners a strong starting point for a hospitality website

## What is included
- Responsive one-page website layout
- Header navigation with mobile toggle
- Hero section with CTA buttons and stats
- Trust strip for quick selling points
- Signature menu cards
- About section
- Gallery section
- Reviews/testimonials section
- FAQ section
- Call-to-action banner
- Contact/enquiry section
- Footer
- Modular CSS structure
- Basic JavaScript for navigation toggle and footer year

## Folder contents
- `src/index.html` — the main website page
- `src/assets/css/` — all styling files, split into base, layout, components, and sections
- `src/assets/js/main.js` — small interactive behaviours
- `src/assets/images/` — place your final images here
- `docs/customisation-guide.md` — notes for editing colours, copy, images, and structure
- `docs/README.md` — this overview file

## Best suited for
- Restaurants
- Bistros
- Wine bars
- Cafes
- Hospitality landing pages
- Freelancer or agency starter projects

## Important notes
- This is a static template. The contact form is styled and ready to connect, but it does not submit anywhere by default.
- Placeholder panels are included so real venue photography can be added later.
- Colours, typography, and content can be updated without changing the overall structure.

## Getting started
1. Open `src/index.html` in a browser.
2. Replace the placeholder content with your venue or client content.
3. Add your images to `src/assets/images/`.
4. Update colours and spacing in the CSS files if needed.
5. Connect the enquiry form to your preferred form handling service if required.

## Deployment
This template can be uploaded to any static hosting platform, including Netlify, Vercel, GitHub Pages, or standard shared hosting.
