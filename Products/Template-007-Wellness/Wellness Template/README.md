# Solace Wellness

Solace Wellness is a clean, modern one-page HTML/CSS/JS template designed for wellness businesses such as coaches, studios, therapists, recovery services, and holistic practitioners.

## Who it is for
- Wellness coaches
- Therapists and recovery businesses
- Yoga, pilates, or mindfulness studios
- Client-focused wellness service providers

## Sections included
- Sticky navigation
- Hero with stats
- Trust strip
- Services
- About
- Gallery
- Process
- Testimonials
- FAQ
- CTA banner
- Contact form
- Footer

## Tech stack
- HTML
- CSS
- JavaScript

## File structure
- `index.html`
- `assets/css/style.css`
- `assets/js/main.js`
- `assets/images/`
- `docs/customisation-guide.md`
- `docs/final-checklist-review.md`

## Customisation
Edit `index.html` to change:
- brand name and initials
- section copy
- placeholder visuals
- contact details
- CTA labels

Edit `assets/css/style.css` to change:
- colours in `:root`
- spacing and container width
- visual placeholder treatments
- button styling

## Deployment
This is a static template and can be deployed to Netlify, Vercel, GitHub Pages, or any standard static host.
