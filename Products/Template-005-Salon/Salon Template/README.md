# Velora Salon

Velora Salon is a static one-page website template designed for salons, boutique studios, colour specialists, and independent stylists.

## Included

- `index.html` — main landing page
- `assets/css/style.css` — all styling for the template
- `assets/js/main.js` — small mobile navigation script
- `assets/images/` — reserved for real image assets
- `docs/customisation-guide.md` — handoff guide for edits
- `docs/product-checklist-pass.md` — final product pass summary

## Key sections

- Sticky navigation
- Hero with supporting stats
- Trust strip
- Services
- About
- Gallery
- Testimonials
- FAQ
- CTA banner
- Contact form
- Footer

## Notes

This is a static template. The contact form is presentational and should be connected to a backend or form service for live submissions.
