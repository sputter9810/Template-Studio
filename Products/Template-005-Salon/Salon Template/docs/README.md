# Luma Salon — Customer Overview

Luma Salon is a static one-page website template for salons, hair studios, boutique beauty spaces, and solo stylists.

## What this template is for
This template is designed to help service-based beauty brands present themselves professionally online, explain their services clearly, and encourage more enquiries or bookings.

## What is included
- A polished one-page homepage
- Salon-focused hero section
- Services section
- About section
- Gallery section
- Testimonials section
- FAQ section
- Contact section with styled enquiry form
- Mobile navigation
- Modular CSS file structure for easy edits

## Best suited for
- Hair salons
- Boutique beauty studios
- Solo stylists
- Colour specialists
- Small teams wanting a premium online presence

## Technical format
This is a static HTML, CSS, and JavaScript template. It does not require a framework to run.

## Important note
The included contact form is pre-styled but does not send submissions on its own. A form service or backend connection is needed for live enquiries.
