# Lumaform - Static Pilates Studio Website Template

Lumaform is a refined one-page static website template made for boutique pilates studios, reformer spaces, and movement-focused wellness brands that want more intro pack bookings and pricing enquiries.

## What this template is for
Use this template to:
- promote a first-class or intro-pack offer clearly
- explain class styles without overwhelming beginners
- show a mobile-friendly timetable and easy-to-compare pricing
- introduce instructors and trust signals early
- guide visitors toward a booking or package enquiry

## Included sections
- sticky navigation
- hero section with CTA buttons and stats
- trust strip
- class styles section
- schedule section
- pricing section
- about section
- instructor section
- testimonials
- FAQ
- CTA banner
- contact section with form
- footer

## Theme selector
This template includes a built-in theme selector panel with five color schemes: Sand, Sage, Rose, Stone, and Olive.
