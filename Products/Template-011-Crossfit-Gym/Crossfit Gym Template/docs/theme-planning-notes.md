# Theme Planning Notes

## Default direction
- Dark, bold, high-energy
- CrossFit-specific positioning
- Focus on trial bookings, visible coaching, and clear weekly structure

## Theme variants
- Ember: warm orange highlight on a dark charcoal base
- Toxic Lime: aggressive lime accent for a high-intensity look
- Competition Red: red accent with a stronger performance feel
- Steel Blue: cooler athletic palette with a clean contrast
- Chalk: light theme for buyers who want a brighter presentation

## Typography direction
- Barlow Condensed for uppercase display headings
- Inter for readable body copy and UI text
