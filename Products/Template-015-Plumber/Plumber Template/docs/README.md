# Harbor Flow Plumbing - Plumber Website Template

A polished one-page website template built for plumbers who want a practical, trustworthy online presence and more local service calls.

## What this template is for

This template is designed for:
- independent plumbers
- plumbing contractors
- emergency plumbing services
- maintenance-focused trade businesses
- small plumbing teams serving local suburbs

It is especially suited to businesses that want a mobile-first lead generation page with strong click-to-call actions, emergency messaging, service-area coverage, and quote-request forms.

## What is included

- responsive one-page homepage
- mobile navigation
- hero section with quote and click-to-call actions
- trust strip
- emergency plumbing banner
- services section
- why choose us section
- service areas section
- testimonials section
- FAQ section
- contact section with quote form UI
- modular CSS structure for easier editing
- customisation guide for setup and editing

## Before you use it

You will usually want to:
- replace the business name and contact details
- update the suburbs and service area list
- tailor the emergency banner to match actual availability
- customise the service cards to match the plumbing business
- connect the form to your preferred backend or form service

## Technical notes

This is a static template built with:
- HTML
- CSS
- JavaScript

No build tools or frameworks are required. You can open the homepage directly in a browser or upload the `src` folder to your preferred static hosting platform.

## Theme selector

This template includes a built-in theme selector panel with five plumber-friendly colour schemes: Harbor Blue, Copper Line, Slate Flow, Fresh Aqua, and Midnight Brass.

To change the default theme, update the `data-theme` value on the `<body>` tag and the `defaultTheme` value in `src/assets/js/main.js`.
