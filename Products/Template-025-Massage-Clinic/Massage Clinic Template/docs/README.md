# Stillpoint Massage Clinic - Static Massage Clinic Website Template

Stillpoint Massage Clinic is a calm one-page static website template made for remedial massage clinics, sports recovery practices, relaxation-focused studios, and treatment-led wellness businesses that want more bookings and enquiries.

## What this template is for
Use this template to:
- showcase massage treatments with clear, practical copy
- guide visitors toward a first appointment or best-fit session
- reinforce trust with practitioner bios, testimonials, pricing, and clinic details
- make hours, location, and booking paths easy to find on mobile
- support remedial, sports, recovery, and general wellness positioning without extra pages

## Included sections
- sticky navigation
- hero section with CTA buttons and trust stats
- trust strip
- treatments section
- pricing section
- approach and trust section
- practitioner section
- testimonials
- visit section with hours and location
- FAQ
- CTA banner
- booking section with form
- footer
