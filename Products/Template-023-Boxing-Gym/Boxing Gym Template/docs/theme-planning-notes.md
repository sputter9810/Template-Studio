# Theme Planning Notes

## Default direction
- Dark, disciplined, energetic
- Boxing and martial-arts-specific positioning with beginner-friendly credibility
- Focus on trial bookings, structured coaching, and class clarity

## Theme variants
- Championship Red: dark charcoal base with a sharp red accent for the flagship combat-sport look
- Midnight Gold: premium dark palette with warm gold contrast
- Ring Blue: cool arena-inspired palette with stronger blue energy
- Graphite Ice: neutral dark mode option with colder metallic highlights
- Canvas Light: light mode variant for buyers who want a brighter presentation

## Typography direction
- Barlow Condensed for strong display headings
- Inter for readable body copy and UI text
