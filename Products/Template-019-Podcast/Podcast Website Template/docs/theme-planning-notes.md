# Theme Planning Notes

## Default direction
- Dark, editorial, personality-led
- Podcast-specific positioning with strong first-screen relevance
- Focus on listening, subscribing, and sponsor credibility

## Theme variants
- Night Studio: deep indigo base with teal contrast for a premium flagship look
- Deep Current: cooler blue-led palette for business, tech, or interview shows
- Coral Edit: warmer editorial palette for culture, lifestyle, or creator brands
- Signal Gold: rich amber palette for premium, insight-led media positioning
- Paper Light: clean light theme for buyers who want a brighter presentation

## Typography direction
- Fraunces for editorial display headings
- Manrope for readable body copy and UI text
