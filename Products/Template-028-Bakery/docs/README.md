# Oven & Crumb - Static Bakery Website Template

Oven & Crumb is a warm one-page static website template made for artisan bakeries, bread shops, pastry cafes, and neighbourhood baked-goods businesses that want more walk-ins, pre-orders, and catering or wholesale enquiries.

## What this template is for
Use this template to:
- showcase signature breads, pastries, cakes, or seasonal bakes with realistic product-led copy
- encourage customers to visit today, place a pre-order, or enquire about catering and wholesale
- reinforce trust with bakery story, freshness cues, testimonials, and clear opening hours
- make location, pickup timing, and contact details easy to find on mobile
- support bread-led bakeries, pastry specialists, and bakery-cafe hybrids without needing ecommerce in the core pack

## Included sections
- sticky navigation
- hero section with CTA buttons and trust stats
- trust strip
- signature bakes section
- orders and offers section
- bakery story section
- menu highlights section
- testimonials
- hours and location section
- FAQ
- CTA banner
- order and contact section with form
- footer
