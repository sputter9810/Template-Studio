# Curbside Sizzle - Static Food Truck Website Template

Curbside Sizzle is a bold one-page static website template made for food trucks, pop-up kitchens, mobile caterers, and street-food brands that need customers to find the truck quickly, scan the menu easily, and send catering enquiries without friction.

## What this template is for
Use this template to:
- showcase signature menu items with realistic, appetite-led placeholder copy
- help visitors find this week's stops without relying on a live schedule integration
- turn catering and event interest into useful booking enquiries
- reinforce trust with event-fit content, testimonials, and clear contact details
- stay easy to customise for burger trucks, taco trucks, BBQ setups, dessert vans, and similar mobile food brands

## Included sections
- sticky navigation
- hero section with CTA buttons and trust stats
- trust strip
- menu highlights section
- catering offers section
- truck story / brand section
- service-fit section for events and bookings
- testimonials
- weekly route section
- FAQ
- CTA banner
- bookings and contact section with form
- footer
