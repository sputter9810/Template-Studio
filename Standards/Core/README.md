# 00-core Template Pack

This folder contains reusable template files for defining a website production system.

## Included Files
- `production-system.md`
- `project-brief.md`
- `design-system.md`
- `technical-standards.md`
- `content-guide.md`
- `qa-checklist.md`

## How To Use
1. Duplicate this folder for a new website/template project.
2. Fill in each file with your studio rules and the product-specific requirements.
3. Send the filled-out files back into ChatGPT and ask for:
   - a build plan
   - a scaffold
   - section-by-section implementation
   - documentation
   - QA review
4. Keep the files updated as your production system evolves.

## Suggested Prompt
“Use the files in `00-core` as the operating system for this project. Fill them out for a [template type] aimed at [niche/customer]. Then produce the first implementation stage.”

## Notes
These are starter templates only. They are intentionally generic so they can be reused across many website products.
