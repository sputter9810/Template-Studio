# Niche Template Planning Bundle

This bundle includes:

1. **Filled Core v2**  
   Ten filled-out Core v2 planning packs, one for each requested niche.

2. **Component Library v6**  
   An updated library based on your uploaded v5 pack, expanded with additional reusable patterns for the new niches.

## Requested niches
- Calisthenics Gym
- Pilates Studio
- Boxing / Martial Arts Gym
- Day Spa
- Massage Clinic
- Coffee Roaster
- Food Truck
- Bakery
- Fine Dining
- Takeaway

## New library additions in v6
### Components
- hours-list
- menu-highlight
- reservation-note
- badge-strip

### Sections
- hours-location-section
- signature-offers-section

These additions were chosen to better support hospitality, wellness, studio, and local-service templates.
